CIE Colour Match API

Endpoints

CIE LMS Cone Fundamentals with optional logarithmic values & 9 significant values:

  • Path: /LMS
  • Method: GET
  • Output: application/json
  • Mandatory Parameters:
    • 'field_size': The wished field_size between 1.0-20.0, float type.
    • 'age': The wished age between 20-80, float type.
  • Optional Parameters:
    • 'min': Minimum value for domain between 390.0-400.0 nm (default = 390.0), float type.
    • 'max': Maximum value for domain between 700.0-830.0 nm (default = 830.0), float type.
    • 'step_size': Minimum value for domain between 0.1-5.0 (nm) (default = 1.0), float type.
    • 'log10': Activates logarithmic values for function, use by placing log10 into URL.
    • 'base': Activates 9 significant figures for function, use by placing base into URL.
  • Examples:
    • /LMS?field_size=1.0&age=64&min=390.0&max=830.0&step_size=0.5&log10&base
    • /LMS?field_size=2.0&age=23
    • /LMS?field_size=2.0&age=23&log10&max=700&min=400

MacLeod-Boynton ls chromaticity diagram:

  • Path: /LMS-MB
  • Method: GET
  • Output: application/json
  • Mandatory Parameters:
    • 'field_size': The wished field_size between 1.0-20.0, float type.
    • 'age': The wished age between 20-80, float type.
  • Optional Parameters:
    • 'min': Minimum value for domain between 390.0-400.0 nm (default = 390.0), float type.
    • 'max': Maximum value for domain between 700.0-830.0 nm (default = 830.0), float type.
    • 'step_size': Minimum value for domain between 0.1-5.0 (nm) (default = 1.0), float type.
    • 'info': Returns instead sidemeny information containing the elements below, use by placing info into URL.
      • 'norm': Normalization coefficients.
      • 'white': Chromaticity point of Illuminant E.
      • 'tg_purple': Tangent points of purple line.
  • Examples:
    • /LMS-MB?field_size=3.0&age=32&min=390.0&max=830.0&step_size=0.75
    • /LMS-MB?field_size=5&age=21
    • /LMS-MB?field_size=1.5&age=51&max=700&info

CIE cone-fundamental-based XYZ tristimulus functions:

  • Path: /XYZ
  • Method: GET
  • Output: application/json
  • Mandatory Parameters:
    • 'field_size': The wished field_size between 1.0-20.0, float type.
    • 'age': The wished age between 20-80, float type.
  • Optional Parameters:
    • 'min': Minimum value for domain between 390.0-400.0 nm (default = 390.0), float type.
    • 'max': Maximum value for domain between 700.0-830.0 nm (default = 830.0), float type.
    • 'step_size': Minimum value for domain between 0.1-5.0 (nm) (default = 1.0), float type.
    • 'norm': Makes all values returned be normalized (including 'info' below, use by placing norm into URL.
    • 'info': Returns instead sidemeny information containing the elements below, use by placing info into URL.
      • 'trans_mat': Transformation matrix of linear transformation LMS --> XYZ.
  • Examples:
    • /XYZ?field_size=4&age=21&min=390.0&max=830.0&step_size=0.2
    • /XYZ?field_size=4&age=80&min=400.0&norm
    • /XYZ?field_size=1.2&age=21&min=390.0&step_size=2.3&info
